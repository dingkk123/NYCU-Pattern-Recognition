# NYCU Pattern Recognition

Coursework and projects for the **Pattern Recognition** course at National Yang Ming Chiao Tung University.

## Repository Structure

- `hw1` - Linear Regression with closed-form solution and gradient descent
- `hw2` - Linear Classification with Logistic Regression and Fisher's Linear Discriminant
- `hw3` - Ensemble Learning with Decision Tree, Bagging, and AdaBoost
- `hw4` - Deep Learning-based Image Classification

## Skills and Topics

- Python
- NumPy
- PyTorch
- Linear Regression
- Logistic Regression
- Fisher's Linear Discriminant
- Decision Tree
- Bagging
- AdaBoost
- CNN
- Model Evaluation

## Description

This repository includes hands-on implementations and reports for pattern recognition and machine learning assignments, covering regression, classification, ensemble methods, and deep learning.
